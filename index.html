scrivo e push

Procedura per qunado creo una nuova cartella/esercizio

  • Creo Cartella Dell'esercizio
  • Creo i file all'interno della cartella
  • Apro vsCode nella cartella dell'esercizio, poi apro il terminale e lancio il comando: git init
  • Faccio il login su github
  • Clicco sull'icona del profilo
  • Clicco su your repositories
  • Clicco su new
  • Inserisco il nome che deve essere uguale alla cartella che ho creato sul pc
  • Poi clicco crea repository
  • Copio il commando che inizia con git remote add origin "link"
  • Torno sul mio terminale in vs code e incollo il comando sopra e premo invio
  • Creo un nuovo branch, con il commando git checkout -b Master e premo invio
  • Lancio il commando git add .
  • Lancio il commando git commit -m "Messaggio a caso"
  • Lancio il commando git push --set-upstream origin Master
