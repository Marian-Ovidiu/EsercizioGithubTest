Procedura per qunado creo una nuova cartella/esercizio

  • Creo Cartella Dell'esercizio
  • Creo i file all'interno della cartella
  • Apro vsCode nella cartella dell'esercizio, poi apro il terminale e lancio il comando: git init
  • Faccio il login su github
  • Clicco sull'icona del profilo
  • Clicco su your repositories
  • Clicco su new
  • Inserisco il nome che deve essere uguale alla cartella che ho creato sul pc
  • Poi clicco crea repository
  • Copio il commando che inizia con git remote add origin "link"
  • Torno sul mio terminale in vs code e incollo il comando sopra e premo invio
  • Creo un nuovo branch, con il commando git checkout -b Master e premo invio
  • Lancio il commando git add .
  • Lancio il commando git commit -m "Messaggio a caso"
  • Lancio il commando git push --set-upstream origin Master
  • Ora devo dare i permessi a marian
  • Vado sulla mia repo
  • Vado su settings e poi clicco su collaborators
  • Poi clicco su add people e cerco marian e lo aggiungo

Procedura per clonare la repo di qualcun'altro

  • Apro visual code
  • File e poi Open folder e scelgi la cartella degli esercizi generali
  • Vado sul profilo di Marian su github
  • Vado sulla repo che voglio clonare
  • Clicco su code e controllare che sia su Local e https, poi copio il link
  • Quindi nel terminale scrivo git clone e incollo il link che ho copiato
  • Apro file/open folder e scelgo la cartella che ho appena clonato
  • Quindi ora sono libero di fare tutte le mie modifiche e quando ho finito lancio i seguenti commandi:
  • git add .
  • git commit -m "Messaggio"
  • git push
  • Dopo aver eseguito questi 3 commandi, ora il tuo codice si trova online
